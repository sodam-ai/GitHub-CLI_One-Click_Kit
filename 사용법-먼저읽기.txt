========================================
  GitHub CLI 원클릭 키트 - 먼저 읽기
========================================

처음이세요? 아래 3단계만 따라 하세요.

  1) INSTALL.bat  더블클릭   ->  설치
  2) RUN.bat      더블클릭   ->  한국어 메뉴 열림
  3) 메뉴에서 [1] 로그인 -> [1] 브라우저 로그인

----------------------------------------
참고
----------------------------------------
- "Windows가 PC를 보호했습니다" 창이 뜨면
  -> [추가 정보] -> [실행] 을 누르세요.

- "gh 를 찾을 수 없다" 고 나오면
  -> 터미널 창이나 PC를 다시 시작하세요.

- 자세한 설명은 README.md 파일을 보세요.

- 제거하려면 UNINSTALL.bat 을 더블클릭하세요.
  (지우기 전에 설정을 백업할지 물어봅니다.)

## 1. 이 키트는 무엇인가요

**GitHub CLI**(`gh`)는 GitHub가 만든 **공식 프로그램**으로, 웹사이트 대신 컴퓨터에서
명령어로 GitHub를 다루는 도구입니다. 보통은 영어 명령어를 직접 입력해야 합니다.

이 키트는 그 `gh` 를 **쉽게 설치하고**, **한국어 메뉴**로 사용하도록 도와주는
**3개의 실행 파일(.bat)** 묶음입니다.

| 파일 | 하는 일 |
|---|---|
| `INSTALL.bat` | GitHub CLI 를 자동으로 설치합니다 (이미 있으면 최신으로 업그레이드). |
| `RUN.bat` | 한국어 메뉴 제어판을 엽니다 (로그인·저장소·이슈·검색 등 모든 기능). |
| `UNINSTALL.bat` | GitHub CLI 를 안전하게 제거합니다 (지우기 전 설정 백업 가능). |

> **중요:** 이 키트는 `gh` 를 **들고 다니지(포함하지) 않습니다.** 설치할 때
> 공식 출처(winget·Scoop·GitHub 공식 다운로드 페이지)에서 **직접 받아옵니다.**

## 2. 사전 준비물 / 필요 프로그램

설치 전에 아래가 준비되어 있어야 합니다.

- **운영체제:** Windows 10 (22H2 이상) 또는 Windows 11
- **인터넷 연결:** 설치·로그인·대부분의 기능에 필요합니다.
- **GitHub 계정:** 없으면 https://github.com 에서 무료로 가입하세요.
- **설치 도구(둘 중 하나, 없어도 됨):**
  - **winget** — Windows 10 22H2 이상과 Windows 11 에 **기본 포함**되어 있습니다.
  - **Scoop** — 선택 사항입니다. https://scoop.sh
  - 둘 다 없어도 됩니다. 이 키트가 **공식 설치 파일(.msi)을 직접 받아서** 설치할 수 있습니다.

> 따로 미리 깔아야 할 프로그램은 없습니다. `INSTALL.bat` 이 알아서 처리합니다.

## 3. 다운로드 방법

1. 이 키트가 들어 있는 폴더(또는 ZIP 파일)를 받습니다.
2. **ZIP 으로 받았다면** 마우스 우클릭 → **압축 풀기**로 풀어 주세요.
   (ZIP 안에서 바로 실행하면 동작하지 않을 수 있습니다.)
3. 풀어 놓은 폴더 안에 `INSTALL.bat`, `RUN.bat`, `UNINSTALL.bat` 가 보이면 준비 끝입니다.

> **"Windows가 PC를 보호했습니다"(SmartScreen)** 창이 뜨면 → **추가 정보 → 실행**을 누르세요.
> 인터넷에서 받은 파일이라 처음 한 번 뜨는 정상적인 안내입니다.

## 4. 빠른 시작 — 딱 3단계

1. **`INSTALL.bat`** 을 더블클릭 → GitHub CLI 가 설치됩니다.
   (설치가 끝나면 "지금 바로 시작할까요?" 라고 물어봅니다. **Y** 를 누르면 바로 다음 단계로 갑니다.)
2. **`RUN.bat`** 을 더블클릭 → 한국어 메뉴 창이 열립니다.
3. 메뉴에서 **`[17] 쉬운 시작`** 을 고르세요 → **로그인부터 저장소까지 1-2-3 단계로 안내**해 줍니다.

> **로그인은 이렇게 됩니다:** 화면에 **8자리 코드**가 나옵니다. 그 코드를 복사하고,
> 자동으로 열리는 브라우저에 붙여넣은 뒤 **Authorize(승인)** 를 누르면 끝입니다.

직접 하고 싶다면 메뉴에서 `[1] 로그인 관리` → `[1] 로그인 - 브라우저` 를 골라도 됩니다.

## 5. 파일 위치 / 문서 위치

이 키트의 모든 파일은 **하나의 폴더 안**에 있습니다. (폴더 위치는 사용자가 둔 곳입니다.)

**실행 파일**
- `INSTALL.bat` — 설치
- `RUN.bat` — 사용(메뉴 제어판)

**문서**
- `README.md` (이 문서, 한국어) / `README.en.md` (English)

## 7. 안전 안내

- **삭제·제거처럼 되돌릴 수 없는 작업**은 실행 전 반드시 화면에 **`YES`** 를 직접
  입력해야만 진행됩니다. (실수로 지워지지 않게 막아 둔 안전장치입니다.)
- 제거(`UNINSTALL.bat`)할 때는 **설정을 먼저 백업**할지 물어봅니다.
- 이 키트는 사용자의 **비밀번호나 개인정보를 따로 저장하지 않습니다.** 로그인은 GitHub
  공식 절차(`gh`)를 그대로 사용하며, 모든 작업은 **사용자 컴퓨터에서** 이루어집니다.

## 8. 문제·오류가 생기면

- **무엇이든 안 되면** → `RUN.bat` 메뉴에서 **`[16] 문제 해결`** 을 누르세요.
  인터넷·설치·로그인 상태를 **자동 점검**하고 해결 방법을 한국어로 알려줍니다.
- **"gh 를 찾을 수 없다"** 고 나오면 → 터미널 창이나 PC 를 **다시 시작**하세요.
  (설치 직후에는 경로가 반영되는 데 시간이 걸립니다.)
- **설치가 실패하면** → `INSTALL.bat` 을 마우스 **우클릭 → 관리자 권한으로 실행**해 보세요.
  ( `INSTALL.bat` 은 권한이 없으면 자동으로 다시 물어봅니다.)
- **인터넷 오류처럼 보이면** → 와이파이/랜선을 확인하세요. 회사망이면 방화벽일 수 있습니다.

더 자세한 표는 [GUIDE.md 의 "문제·오류 대처"](GUIDE.md) 를 보세요.

## 9. 제거 방법

1. **`UNINSTALL.bat`** 을 더블클릭합니다.
2. 설정을 **백업할지** 물어봅니다. (원하면 `y`)
3. 정말 지울지 확인하기 위해 **`YES`** 를 입력하면 → 로그아웃 → 제거 → 설정/캐시 정리가 진행됩니다.

> 키트 폴더 자체(.bat·문서)는 그대로 남습니다. 필요 없으면 폴더를 휴지통으로 옮기면 됩니다.
